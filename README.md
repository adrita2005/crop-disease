# 🍋 CitrusAI - Citrus Disease Detector

An AI-powered web application to detect citrus plant diseases from leaf photos.

## 🌐 Live Demo
👉 [Click here to try the app](https://crop-disease-sand.vercel.app)

## 🚀 Features
- 📷 Upload or capture leaf photos
- 🔬 Instant AI disease detection
- 💊 Treatment recommendations
- 📄 PDF report download
- 📊 Scan history & analytics
- 🌙 Dark/Light mode

## 🦠 Detectable Diseases
- Canker
- Greening
- Gummosis
- Healthy
- Leaf-miner
- Lemon-butterfly

## 🤖 Tech Stack
| Part | Technology |
|---|---|
| Frontend | React + Vite |
| Backend | FastAPI Python |
| AI Model | MobileNetV2 + TensorFlow |
| Dataset | 19,688 citrus leaf images |
| Deployment | Vercel + Render |
